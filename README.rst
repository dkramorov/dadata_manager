Описание
-----------
Менеджер для работы с Dadata

Установка пакетом
-----------
Для локальной разработки::
    pip install -e packages/dadata_manager

Импорт
-----------
Проверка::
    from managers.dadata_manager import DadataManager


Удаление
-----------
Удалить пакет::
    pip uninstall dadata_manager

Для создания пакета
https://docs.python.org/3.10/distutils/introduction.html#distutils-simple-example
https://docs.python.org/3.10/distutils/sourcedist.html
::
    python setup.py sdist
